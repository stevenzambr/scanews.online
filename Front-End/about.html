About Me

My name is Steven. I created this website to improve my development skills and familiarize myself with Front-end and Back-end development.

Scanews

Scanews is a news headlines web scraping website and currently supports 6 news sites. The shown images and headlines are clickable and redirect to the corresponding news site article. The website is updated automatically approximately every 2 minutes. (Why? See Disclaimer below). The most recent update to the website can be seen on the bottom right section of the Scanews Home page.

Original and Keyword Setting (for images)

This setting changes the shown images for the headlines. The "Original" option shows the original image from the news site, the "Keyword" option uses natural language processing to summarize the headlines into keywords, which are used to search the web for a fitting image. Keep in mind when using the keyword option the searched images can vary in relevance to the headline of the news site. This feature was added due to the restrictions some news site set on the web scraping of their images. (Original images are not available for "CNBC" and "NBC" headlines).

How does it work?

Scanews was built from the ground up, without the help of a website building service.
For the Back-End Python and JavaScript were used to handle the web scraping, data structure creation, data transfer to the website server and reading of the data. Used libraries include Requests, Beautiful Soup, Gensim, Paramiko and jQuery .
For the Front-End HTML, CSS and Javascript were used alongside with Bootstrap 5.0 as CSS Framework.

Disclaimer

Scanews does not own any of the shown text and pictures. Scanews does not and will never provide complete readable news articles on this website. All provided headlines redirect to the source web site of the article. Every shown image on Scanews has a source that can be accessed by clicking the i button on the bottom right corner of every image.

Regarding Web Scraping

Scanews takes an ethical approach to web scraping by using a very slow request rate of 1 request per two minutes to the supported news sites. This is done to reduce the used bandwidth of the requested sites.
